# Кейс: УСН «доходы» + маркетплейс + цифровые товары (поставщик)

Первый **детерминированный леджер** строится на твоём личном сценарии: три независимых источника — **маркетплейс**, **банк**, **поставщик цифровых товаров**.

Спецификация: [SPEC.md](./SPEC.md).  
Примеры событий: [examples/sample-events.jsonl](./examples/sample-events.jsonl).

**Важно:** налоговая база УСН и момент дохода — вопрос **версии правил** (`rules_version`) и консультанта; леджер гарантирует **воспроизводимость** и **сверку потоков**, а не «одобрение ФНС».
